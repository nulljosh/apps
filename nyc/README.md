# nyc

![version](https://img.shields.io/badge/version-v1.0.0-blue)

Times Square Survival: macOS colony sim set in Times Square. SpriteKit, Swift 6, landscape.

## Features

- Colonist AI with health bars, directives, XP/leveling, traits
- Weapons and combat system
- Building placement and demolition
- 3-slot save system with auto-save
- Interactive tutorial
- SF Pro fonts, settings overlay

## Run

```bash
xcodegen generate && open TimesSquareSim.xcodeproj
```

Requires Xcode 16+, macOS 15.0+, xcodegen.
