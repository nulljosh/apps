# wiretext-macos

Unicode wireframe tool, native macOS SwiftUI.

## Stack: SwiftUI, macOS 14+, xcodegen, Swift 6

## Key files
- Sources/Models/CanvasModel.swift — grid state
- Sources/Models/ComponentType.swift — 23 presets
- Sources/Views/CanvasView.swift — NSTextView canvas
- Sources/Views/ToolbarSidebar.swift — component palette

## Build: xcodegen generate && open Wiretext.xcodeproj
