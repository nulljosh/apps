# Joshua Adam Trommel

![version](https://img.shields.io/badge/version-v4.0.0-blue)

Personal life summary and timeline for therapy. 32 text sections interleaved with 24 visual elements: charts, pull quotes, a stats dashboard, and a geography map. True 50/50 split between prose and visuals.

## Features

- Dual timeline: horizontal SVG (desktop) + vertical HTML (mobile)
- 18 inline SVG charts: stability, events by phase, aggression, triggers, sleep quality, loss clustering, sensory heatmap, diagnosis gap, relationships, attachment spectrum, social circle, housing, substance trajectory, coping mechanisms, screen time, daily routine, financial timeline, strength bars
- 4 pull quotes: key sentences isolated for impact
- Key figures stats grid (9 numbers: diagnosis gap, aggression years, first therapist age, current age, friends nearby, rehab stays, daily bong tokes, IQ, weed help/hurt ratio)
- Geography map: BC locations + off-map (NYC, Florida, Hawaii)
- Auto light/dark mode
- Scroll fade-up animations (IntersectionObserver)
- Print CSS for PDF export
- Color-coded events: red (crisis), black (event), green (positive)
- Safe area support for notched devices
