# Joshua Adam Trommel

![version](https://img.shields.io/badge/version-v3.0.0-blue)

Native iOS companion for the [Life](https://github.com/nulljosh/apps/tree/main/life) therapy summary. SwiftUI, iOS 17+, static data, automatic light/dark mode.

## Screenshot

<img src="screenshots/main.png" width="300">

## Features

- Full-viewport paging scroll with text + visuals combined per page
- 18 Swift Charts: stability, events by phase, aggression, triggers, diagnosis gap, relationships, social circle, housing, coping mechanisms, daily routine, sensory heatmap, sleep quality, financial timeline, substance use, strengths, boundaries, masking energy budget, same body
- 4 pull quotes, 12-stat key figures grid, geography map
- Vertical timeline with color-coded categories (crisis, event, forward)
- 32 content sections matching the web version
- System fonts, minimal design

## Build

```bash
xcodegen generate
open Life.xcodeproj
```
