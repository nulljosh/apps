# Acupuncture macOS

![version](https://img.shields.io/badge/version-v1.0.0-blue)

Native macOS companion for the acupuncture/reflexology web app. Sidebar navigation, three-column layout for acupuncture points, interactive SVG body maps, session tracker.

## Run

```bash
xcodegen generate && open Acupuncture.xcodeproj
```
