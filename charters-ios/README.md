# Charters iOS

![version](https://img.shields.io/badge/version-v0.0.1-blue) ![platform](https://img.shields.io/badge/platform-iOS%2017%2B-lightgrey) ![license](https://img.shields.io/badge/license-MIT-green)

Native iOS companion for [Charters](../charters/) — Constitutional Rights Reference. Browse and compare rights across 15 constitutions worldwide.

---

## Features

- Country list with region grouping and tag-based category filtering
- Country detail view with full article browsing
- Compare mode — side-by-side rights comparison for two countries
- Article detail with full text
- User profiles and case tracking

## Build

```bash
cd apps/charters-ios
xcodegen generate
open Charters.xcodeproj
```

Target: iPhone + iPad (iOS 17+). Default simulator: iPhone 17 Pro.
