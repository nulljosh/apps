# wiretext web

Unicode wireframe tool. Vite + React 19.

## Key files
- src/lib/presets.js — 23 component templates
- src/lib/engine.js — grid ops (stamp, render, pixelToGrid)
- src/components/Canvas.jsx — click-to-place canvas
- src/components/Toolbar.jsx — component palette
- src/components/Inspector.jsx — cursor pos + layers

## Run: npm install && npm run dev
