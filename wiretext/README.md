# wiretext

Unicode wireframe design tool. Web app.

**Version**: 1.0.0 | **Stack**: Vite, React 19

## Run
npm install && npm run dev

## Features
- 23 component presets (Button, Input, Card, Modal, Table, etc.)
- Click-to-place on monospace canvas
- Export as .txt or copy to clipboard
- Undo/redo, layer panel
