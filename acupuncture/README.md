# Acupuncture & Reflexology

![version](https://img.shields.io/badge/version-v1.0.0-blue)

Interactive body maps and pressure point reference for acupuncture and reflexology. Foot and hand reflexology with clickable SVG zones, 35+ acupuncture points across 11 meridians, symptom finder, and session tracker.

## Features

- Interactive foot reflexology map (16 zones)
- Interactive hand reflexology map (11 zones)
- Acupuncture points reference by meridian
- "What Hurts?" symptom finder (12 conditions)
- Session tracker with localStorage

## Run

```bash
npm install && npm run dev
npm run build
```
