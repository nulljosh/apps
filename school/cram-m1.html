Unit 1

Sequences & Series

Arithmetic

aₙ = a₁ + (n−1)d Sₙ = n/2 · (a₁ + aₙ) Sₙ = n/2 · (2a₁ + (n−1)d)
  • d = common difference = a₂ − a₁
  • Find n: isolate n after subbing in known values

Geometric

aₙ = a₁ · r^(n−1) Sₙ = a₁(1 − rⁿ) / (1 − r) S∞ = a₁ / (1 − r) [only if |r| < 1]
  • r = common ratio = a₂ / a₁
  • Find n: aₙ = a₁rⁿ⁻¹ → log both sides

Convergent vs Divergent

|r| < 1 → converges (S∞ exists)

|r| ≥ 1 → diverges (NO S∞)

r = −0.99 → |r| = 0.99 < 1 → CONVERGES

Sigma Notation

Σᵢ₌₁ⁿ (a₁ + (i−1)d) ← arithmetic Σᵢ₌₁ⁿ a₁·r^(i−1) ← geometric

To find S from σ to n terms: Sₙ − Sσ₋₁

Common Errors to Avoid

  • U1: Check sign of r. r = −0.99 → |r| = 0.99 < 1 → STILL CONVERGES.
  • U1: S∞ only exists when |r| < 1. If |r| ≥ 1, write "divergent, no sum."
  • U2: h shifts RIGHT when positive. y = f(x−3) shifts 3 units RIGHT, not left.
  • U2: Horizontal stretch = 1/|b|. If b=2, graph compresses horizontally by ½.
  • U3: Always factor out GCF before attempting other methods.
  • U3: Sum of cubes middle sign is MINUS: (a+b)(a²−ab+b²).
  • U3: Diff of cubes middle sign is PLUS: (a−b)(a²+ab+b²).
  • U3: For messy trinomials, verify by mentally multiplying back.
Unit 3

Polynomials

Factoring — Always GCF First

a² − b² = (a−b)(a+b) a³ + b³ = (a+b)(a²−ab+b²) a³ − b³ = (a−b)(a²+ab+b²)

Simple Trinomial (a = 1)

x² + bx + c = (x+p)(x+q)

Find p, q: p + q = b and p · q = c

AC Method (Messy, a ≠ 1)

  • 1. Multiply a·c (target product)
  • 2. Find two #s: multiply to ac, add to b
  • 3. Split middle term using those two #s
  • 4. Factor by grouping
6x²−13x+6: ac=36, find −9,−4 → 6x²−9x−4x+6 → (3x−2)(2x−3)

Grouping (4 Terms)

ax+ay+bx+by = a(x+y)+b(x+y) = (a+b)(x+y)

Substitution (Biquadratic)

2x⁴+9x²+9: let u=x² 2u²+9u+9 = (2u+3)(u+3) → (2x²+3)(x²+3)
Unit 3 cont.

Theorems & Graphs

Remainder & Factor Theorems

  • P(a) = remainder when P(x) ÷ (x−a)
  • (x−a) is a factor ⟺ P(a) = 0

Rational Root Theorem

possible roots = ±(factors of constant) / (factors of leading coeff)

Test with synthetic division. If P(r)=0, (x−r) is a factor.

Polynomial Graphs

  • Degree n → max n x-intercepts, max n−1 turns
  • Leading coeff +, even degree → both ends up
  • Leading coeff −, odd degree → left up, right down
  • Odd multiplicity → crosses x-axis at root
  • Even multiplicity → bounces off x-axis at root
Unit 2

Transformations

General Form

y = a·f(b(x − h)) + k
  • a — vertical stretch by |a|; a < 0 reflects over x-axis
  • b — horizontal stretch by 1/|b|; b < 0 reflects over y-axis
  • h — horizontal shift RIGHT by h
  • k — vertical shift UP by k

Mapping Rule

(x, y) → (x/b + h, ay + k)

Apply b first (divide x), then shift by h. Don't mix up order.

Inverse Functions

  • Swap x and y, then solve for y
  • Notation: f⁻¹(x)
  • Graph: reflection over line y = x
  • Domain of f = Range of f⁻¹ and vice versa

Reflections

  • Over x-axis: (x, y) → (x, −y) → multiply by −1 outside
  • Over y-axis: (x, y) → (−x, y) → replace x with −x inside
  • Over y = x: swap x and y (inverse)

Domain & Range Shifts

D: [d₁/b + h, d₂/b + h] R: [a·r₁ + k, a·r₂ + k]

Quick Reference — Key Facts

Arithmetic Trick

If you know a₁, aₙ, and n: Sₙ = n(a₁+aₙ)/2

Geometric Trick

To find missing term: aₙ/a₁ = r^(n−1), take nth root

Transformation Quick Check

y = 2f(3(x−1)) + 4

→ stretch V×2, compress H×⅓, shift R1, shift U4

Factoring Decision Tree

  1. GCF?
  2. 2 terms: diff of squares? cubes?
  3. 3 terms, a=1: simple trinomial
  4. 3 terms, a≠1: AC method
  5. 4 terms: grouping
  6. 4+ terms or degree 4: substitution or rational roots
