# wiretext-ios

Unicode wireframe tool, native iOS SwiftUI.

## Stack: SwiftUI, iOS 17+, xcodegen, Swift 6

## Key files
- Sources/Models/CanvasModel.swift — grid state (80x50)
- Sources/Models/ComponentType.swift — 23 presets
- Sources/Views/CanvasScrollView.swift — UITextView canvas + tap handler
- Sources/Views/ComponentPickerView.swift — bottom sheet picker

## Build: xcodegen generate && open WiretextIOS.xcodeproj
